# Smart P&ID

Online engineering application for interactive P&ID networks, engineering objects, source-document traceability, maintenance data, and future AI-assisted search.

## Current build target

- Service Water network
- Smart vector P&ID
- Pump 501-P-0105 A/B/C/D
- A1A PMS integration
- VMS integration
- Pump datasheet, GA and cross-sectional data

## Planned stack

- Next.js
- TypeScript
- Supabase
- Vercel
